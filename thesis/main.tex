\chapter{Постановка задачи}

Целью настоящей работы является \textbf{автоматизация разработки механизмов композиции в предметно-ориентированных языках}. Для достижения поставленной цели необходимо решить следующие задачи:
\begin{itemize}
\item Разработать автоматизированный метод расширения предметно-ориентированных языков поддержкой типизированных макроопределений. 
\item Разработать автоматизированный метод расширения предметно-ориентированных языков поддержкой аспектов. 
\item Формально описать разработанные методы и обосновать их корректность.
\item Продемонстрировать применение разработанных методов, применив их к предметно-ориентированному языку, полезному для решения практических задач.
\end{itemize}
